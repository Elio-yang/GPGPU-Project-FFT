
《本README说明该程序的编译运行方式》
------------------gFFT/-------------------------
LINUX:
1.      按照CUDA 和 CUDNN ，确保nvcc命令可以使用
2.      输入make
3.      ./gFFT  [阶数1]   [阶数2]  [线程数]
WIN:
1.      配置好VS
2.      装入代码点击运行即可
---------------------------------------------------

------------------OM_rFFT/------------------------
WIN/LINUX:
1.       安装好cmake(>3.10)
2.       新建build文件夹
3.        cd build
4.        cmake ..    [可自行配置生成的makefile]
5.        make
6.        OM 为固定测试无需命令行参数 
7.         ./rFFT  [阶数1]   [阶数2]
------------------------------------------------------
